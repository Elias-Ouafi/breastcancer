# Breast Cancer Detection Project

The project goal is to help doctors and nurse identify breast cancer at the earliest time possible. It won't be used alone but in parallel with doctor advice.
At the end we need an app that will excpect an mri file in input and show to docotrs the probability of breast cancer at it's earliest in the output.
To do that the project will focus on developing machine learning models for breast cancer detection using medical imaging datasets. 

The project currently uses two major datasets:
1. CBIS-DDSM (Curated Breast Imaging Subset of DDSM)
2. BreakHis (Breast Cancer Histopathological Database)

## Prerequisites

- Python 3.8 or higher
- TCIA API key (for CBIS-DDSM dataset)
- Kaggle account (for BreakHis dataset)
